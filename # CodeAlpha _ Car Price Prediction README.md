# CodeAlpha - Car Price Prediction

## Objective
Predict the selling price of a used car using Machine Learning.

## Dataset
Car data dataset containing:
- Car Name
- Year
- Present Price
- Selling Price
- Fuel Type
- Transmission
- Owner
- Driven Kms

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Model Used
Random Forest Regressor

## Results
- MAE: 0.639
- RMSE: 0.969
- R² Score: 0.959

## Conclusion
The model achieved approximately 96% accuracy in predicting car prices.
